# MIxS Version History Research

Static reference documentation for all known MIxS versions and releases.

## Contents

- `version_registry.yaml` - Machine-readable catalog of all versions with metadata
- `mixs_version_timeline.md` - Human-readable timeline with methodology notes

## Purpose

These files document the historical record of MIxS releases from 2005 to present. They are reference material, not programmatic dependencies of the diff tooling.

## Data Sources

- GitHub releases API (`gh release view`)
- xlsx/docx internal metadata extraction
- SourceForge archives
- Publication DOIs (Field 2008, Yilmaz 2011)
